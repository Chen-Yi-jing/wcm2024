# 網際內容管理課程回顧

學員簡報與自評 - 請製作一個帶有說明字幕的期中報告影片, 放入個人的 wcm2024 影片, 以便在 w9 期中考週進行簡報與自評．

申請 Github 帳號: 目的是希望針對網站內容改版的每一個版本都能有效取得

申請 Replit 帳號: 可以在網頁瀏覽器或手機 App 修改網站內容

學習 Brython 程式: 可以直接在網頁上執行與產品設計有關的數值運算
